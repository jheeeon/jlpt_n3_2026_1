# Screen Flow

## Main

첫 화면은 단어장 중심이다.

표시:

- 시험일
- 시험일까지 남은 날짜
- 암기 완료 수
- 즐겨찾기 수
- 오늘의 단어 버튼
- 문법 학습 버튼

`오늘의 단어`는 현재 화면의 활성 메뉴다.

`문법 학습`은 비활성 버튼이다. 클릭해도 기능 동작이 없다.

## Vocabulary

오늘 날짜와 공개 진행 상태를 기준으로 30개 단어를 보여준다.

단어 카드:

- 일본어 표기
- 읽기
- 한국어 의미
- priority
- 검수 필요 표시
- 즐겨찾기 버튼
- 암기 완료 버튼

현재 데이터에는 `needs_review` 항목이 없으므로 검수 필요 표시가 보이지 않는 것이 정상이다. 이 표시는 이후 데이터 재생성 과정에서 검수 대상이 다시 생겼을 때를 위한 UI다.

## Empty/Error

fallback은 없다.

Supabase table 또는 seed가 없으면 SQL 실행 안내를 보여준다.
